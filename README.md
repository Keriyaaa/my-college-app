# MyCollegeApp

Мобильное приложение для студентов и преподавателей колледжа КАИТ №20. Разработано на React Native с использованием TypeScript. Приложение предоставляет удобный интерфейс для навигации по учебной информации и имеет несколько цветовых тем для персонализации.

## Основной функционал

- Навигация по основным разделам приложения
- Поддержка нескольких цветовых тем (оранжевая, синяя, зеленая, фиолетовая)
- Адаптивный интерфейс для различных устройств
- Построение на компонентной архитектуре React Native

## Технологический стек

- **React Native** — фреймворк для кроссплатформенной разработки
- **TypeScript** — типизированный JavaScript для надежности кода
- **React Navigation** — навигация между экранами
- **Metro** — сборщик JavaScript

## Установка и запуск

### Требования

Перед началом убедитесь, что у вас установлены:
- Node.js (последняя стабильная версия)
- npm или Yarn
- Android Studio / Xcode (для эмуляции)
- CocoaPods (для iOS)

### Установка зависимостей

```bash
npm install
# или
yarn install
```

### Запуск Metro (сборщик JavaScript)

```bash
npm start
# или
yarn start
```

### Запуск на Android

```bash
npm run android
# или
yarn android
```

### Запуск на iOS

Для iOS сначала установите зависимости CocoaPods:

```bash
bundle install
bundle exec pod install
```

Затем запустите приложение:

```bash
npm run ios
# или
yarn ios
```

## Структура проекта

Основной код приложения находится в папке `src/`, где реализованы экраны и компоненты. Проект изначально создан с помощью `@react-native-community/cli` и следует стандартной архитектуре React Native.
